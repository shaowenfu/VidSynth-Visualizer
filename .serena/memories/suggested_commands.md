# Suggested Commands

## Project-Specific Commands
*   **Install Dependencies:** `npm install`
*   **Run Development Server:** `npm run dev`
*   **Build for Production:** `npm run build`
*   **Preview Production Build:** `npm run preview`

## Environment Variables
*   Ensure a `.env.local` file exists in the root directory with `GEMINI_API_KEY=your_api_key_here`.

## Common System Utilities (Linux)
*   **List files:** `ls -la` (for detailed list including hidden files)
*   **Change directory:** `cd <directory_name>`
*   **Search file content:** `grep -r "pattern" .` (recursive search)
*   **Find files:** `find . -name "*.tsx"` (find all TypeScript React files)
*   **Version Control:** `git status`, `git add .`, `git commit -m "message"`, `git pull`, `git push`
